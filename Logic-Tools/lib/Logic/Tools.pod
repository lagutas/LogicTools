=head1 SYNOPSIS

=pod

    my $tools=Logic::Tools->new(
                                    config_file	=>	'/etc/kamailio/kamailio-tools.ini',
				                    lock_file	=>	$lock_file,
				                    runas_user	=>	$runas_user,
				                    logfile		=>	'/var/log/dialog_from_kamctl.log',
                                    sounds_dir  =>  '/var/lib/asterisk/sounds/ru'
                                );
    
    my $db_host=$tools->read_config( 'dialog_from_kamctl', 'db_host');
    
    #проверка не запущен ли этот процесс
    $tools->check_proc();
    
    #запускаем как демон
    $tools->start_daemon()
    
    $tools->logprint("info","диалог:  $state $callid $from_uri $to_uri ");

    # Воспроизведение даты и времени
    $AGI = new Asterisk::AGI;
    %input = $AGI->ReadParse();

    $tools->AGIDateSpeach($AGI, "10", "10");
    $tools->AGITimeSpeach($AGI, "23", "45");

    
=cut

=head1 EXPORT

=head1 SUBROUTINES/METHODS

=over 4

=item DecompositionNumber (Number)

Данная функция используется для разложения минут или часов, т.е. передается исходное число, на выходе массив состоящий из разложения этого числа.
например, передаем число 45, на выходе массив их двух элементов 40 и 5.

=back

=head1 constructor

=over 4

=item new ( %ARGS )

=over 8

=item config_file

Путь до файла с конфигами на что-либо

=item lock_file

Файл для блокировки при запуске приложения как демона, в этот файл записываедся текущий pid процесса

=item runas_user

пользователь под которым запускать приложение

=item logfile файл для сохранения лога

=item sounds_dir 

Каталог звуковых файлов

=back

=back

=head1 METHODS

=over 4

=item read_config ( SECTION, PARAM )

метод для чтения параметров из конфигурационного файла config_file

возвращает занчение параметра указанной секции

=over 8 

=item SECTION - конфиг файл состоит из секций, в квадратных скобках указано название секции

=item PARAM	- внутри секции параметрам соответствует значение

=back 

=item check_proc()

проверяет процесс по lock_file на то что он уже запущен, если не запущен возвращает 1, если запущен выдает ошибку в STDERR и завершается

=item logprint ( LOGLEVEL, MESSAGE )

записывает сообщение MESSAGE в лог файл logfile, с уровнем LOGLEVEL

=over 8

item LOGLEVEL - может принимать следующие значения:

=over 12

trace
debug
info
notice
warning
error
critical
alert
emergency

=back

=back

=item start_daemon

Запускает процесс как демон под пользователем runas_user, создает файл блокировки lock_file

=back

=item AGIDateSpeach ( AGI, Day, Month)

В качестве параметров передается уже созданный объект $AGI, день и месяц. Также должен быть указан параметр в конструторе sounds_dir (корневой каталог звуковых файлов),
Имена файлов дня и месяца должны соответствовать передаваемым значениям.
Внимательно проверяйте пути в Playback'ах!

=item AGITimeSpeach ( AGI, Hour, Minute)

В качестве параметров передается уже созданный объект $AGI, час и минуты. Также должен быть указан параметр в конструторе sounds_dir (корневой каталог звуковых файлов),
Имена файлов часа и минут должны соответствовать передаваемым значениям. Так как имеются не все звуковые файлы чисел, то используется функция DecompositionNumber.
Внимательно проверяйте пути в Playback'ах!
